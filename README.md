# CallTaxi
A Modern HTML5/jQuery/CSS3 Theme | Live demo: https://goo.gl/dbFGdp

## Description

This template has been designed for Taxi companies who wish to have a modern website.

* One page
* SVG
* Jquery 
* HTML5
* CSS3 Animation
